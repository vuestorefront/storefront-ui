# component-description
Round hover-animated component with an icon, e.g. usable for major navigation/action items.

# common-usage
<br>
<SfCircleIcon icon="home" />

```html
<template>
  <a href>
    <SfCircleIcon icon="home" />
  </a>
</template>
<script>
import { SfCircleIcon } from "@storefront-ui/vue";

export default {
  components: {
    SfCircleIcon
  }
};
</script>
```
